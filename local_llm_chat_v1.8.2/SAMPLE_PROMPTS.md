# LLMとの会話 サンプルプロンプト集

**JRC 2026 ハンズオンセミナー配布資料** — Local LLM Chat v1.8.0

このプロンプト集は、ハンズオン中にコピー＆ペーストしてお試しいただけるサンプルです。
一般的な内容から専門的な内容へ段階的に進む構成になっています。

## 1. 基本的な質問（ウォームアップ）

まずはシンプルな質問で、ローカルLLMの応答を体験してみましょう。

### 1-1. 一般知識

```
放射線科で使われる主なモダリティを5つ挙げて、それぞれの特徴を1行で説明してください。
```

医療AIの活用が期待されている分野を5つ教えてください。
```

## 2. 文章のリライト・校正

日常業務で使える汎用的なテキスト処理です。

### 2-1. 文章を簡潔にする

```
以下の文章をもっと簡潔にしてください：

「本検査は、X線を用いて体内の断層画像を撮影するものであり、検査時間は概ね10分から15分程度を要し、造影剤を使用する場合にはアレルギー反応等の副作用が生じる可能性があるため、事前に同意書への署名をお願いしております。」
```

### 2-2. 敬語の調整

```
以下の文章を、患者さん向けのやさしい表現に書き換えてください：

「造影CT検査において、非イオン性ヨード造影剤を経静脈的に投与し、動脈相・門脈相・平衡相の3相で撮影を実施する。」
```

---

## 3. 翻訳

### 3-1. 日本語→英語

```
次の文を英語に翻訳してください：

「造影CTにて肝右葉に早期濃染を示す腫瘤を認めます。門脈相ではwashoutを呈しており、肝細胞癌が疑われます。」
```

### 3-2. 英語→日本語

```
次の英文を日本語に翻訳してください：

"A well-defined, round, homogeneous lesion was identified in the right lower lobe of the lung on the chest CT. The lesion measures approximately 12mm in diameter and demonstrates no contrast enhancement."
```

## 4. 画像所見の記載支援

放射線科レポートの下書き作成に活用できます。

### 4-1. 胸部CT所見

```
胸部CTで右肺上葉に径15mmのすりガラス影を認めました。周囲に気管支拡張や胸膜陥入像はありません。所見文を書いてください。
```

### 4-2. 頭部MRI所見

```
頭部MRIで左側頭葉内側にFLAIRで高信号を示す領域を認めます。拡散強調像で高信号、ADCマップで低信号です。所見文と考えられる診断を記載してください。
```

### 4-3. 正常所見テンプレート

```
腹部造影CTの正常所見テンプレートを作成してください。以下の項目を含めてください：
- 肝臓、胆嚢、膵臓、脾臓、腎臓
- 大動脈、下大静脈
- 腹腔内リンパ節
- 腹水の有無
```

## 5. 鑑別診断

### 5-1. 肝腫瘤

```
50歳女性。造影MRIで肝S6に径3cmの腫瘤を認めます。
- T1WI: 低信号
- T2WI: 高信号
- 動脈相: 濃染
- 門脈相: washout
- 肝細胞相: 低信号

鑑別診断を3つ挙げて、それぞれの根拠を説明してください。
```

### 5-2. 肺結節

```
65歳男性、喫煙歴あり。胸部CTで右肺下葉に径20mmの充実性結節を認めます。辺縁は不整で、スピキュラを伴っています。胸膜陥入像あり。縦隔リンパ節腫大なし。

鑑別診断と推奨される次のステップを教えてください。
```

## 7. 論文・文献関連

### 7-1. アブストラクト要約

```
以下のAbstractを日本語で要約してください。要点を箇条書きで示してください：

"Purpose: To evaluate the diagnostic performance of deep learning-based computer-aided detection (DL-CAD) for pulmonary nodule detection on low-dose chest CT in a lung cancer screening setting. Materials and Methods: This retrospective study included 1,200 low-dose chest CT examinations. DL-CAD performance was compared with that of experienced radiologists. Sensitivity, specificity, and area under the receiver operating characteristic curve (AUC) were calculated. Results: DL-CAD achieved a sensitivity of 94.2% and specificity of 88.7%, with an AUC of 0.95. The mean reading time was reduced by 35% when radiologists used DL-CAD as a second reader."
```

### 7-2. 研究デザインの相談

## 8. プリセットプロンプトの活用

Local LLM Chatには、よく使うプロンプトを📋ボタンからワンクリックで呼び出せる「プリセット」機能があります。以下のプリセットが標準搭載されています。

| プリセット | 使い方の例 |
|------------|------------|
| 💻 プログラミング | 「Pythonで画像をリサイズするコードを書いてください」 |
| ✏️ 文章作成・校正 | 「以下の文章を校正してください：〜」 |
| 🌐 翻訳 | 「次の文を英語に翻訳してください：〜」 |
| 💡 ブレインストーミング | 「放射線科の業務効率化のアイデアを5つ」 |
| 🏥 疾患解説 | 「肺塞栓症」と入力するだけで構造化された解説を生成 |
| 💊 鑑別診断 | 臨床・画像情報を入力すると鑑別リストを生成 |
| 📄 文章要約 | 長文を貼り付けて要約を生成 |
| 📝 論文査読 | 研究内容を貼り付けると査読コメントを生成 |
| 📈 統計解析 | データの種類と目的を入力すると適切な手法を提案 |
| ✉️ 英文メール作成 | 相手・用件・トーンを入力すると英文メールを生成 |

### プリセットの使い方

1. 入力欄の左にある「📋」ボタンをクリック
2. 使いたいプリセットを選択
3. テンプレートが入力欄に挿入される
4. 【 】の部分に具体的な内容を入力して送信

## 11. プロンプトエンジニアリング — フレームワーク活用例

プロンプトの書き方を少し工夫するだけで、LLMの回答品質は大きく変わります。
ここでは代表的なフレームワークと、放射線科での具体例を紹介します。

### フレームワーク一覧

| フレームワーク | 構成要素 | 特徴 |
|---------------|----------|------|
| **RTF** | Role / Task / Format | シンプルで汎用的 |
| **RICE** | Role / Instruction / Context / Example | 文脈と例示で精度向上 |
| **Chain of Thought** | 段階的に考えさせる | 推論タスクに強い |
| **Few-shot** | 入出力の例を提示 | 出力形式を正確に制御 |
| **制約付きプロンプト** | 条件・制約を明示 | 不要な出力を排除 |
| **PREP法** | Point / Reason / Example / Point | 説得力のある説明・報告 |
| **PDCA サイクル** | Plan / Do / Check / Act | 改善・業務プロセス設計 |
| **SWOT 分析** | Strengths / Weaknesses / Opportunities / Threats | 意思決定・戦略検討 |
| **6つの思考の帽子** | 白/赤/黒/黄/緑/青の視点 | 多角的・網羅的な分析 |
| **無限議論** | 司会+複数専門家のロールプレイ | 多角的な議論で結論を導く |

---

### RTF（Role / Task / Format）

役割・タスク・出力形式を明示するシンプルなフレームワークです。

```
【Role】あなたは放射線診断専門医です。
【Task】以下の胸部CT所見から、放射線科レポートの所見欄を作成してください。
【Format】「所見」と「印象」に分けて記載してください。

右肺下葉S8に径25mmの充実性結節あり。辺縁不整、スピキュラ(+)。
右肺門リンパ節に短径12mmの腫大あり。胸水なし。
```

---

### RICE（Role / Instruction / Context / Example）

文脈情報と出力例を加えることで、より精度の高い回答を引き出します。

```
【Role】あなたは放射線科のレポート作成を支援するアシスタントです。

【Instruction】以下の臨床情報と画像所見から、鑑別診断を3つ挙げてください。
各診断には自信度（%）と根拠を1行で記載してください。

【Context】
- 患者: 72歳男性、喫煙歴40年
- 主訴: 2ヶ月間の体重減少と血痰
- 画像: 胸部造影CTで右肺上葉に径35mmの不整形腫瘤、縦隔リンパ節腫大あり

【Example（出力形式の例）】
1. ○○○（自信度: 80%）— 根拠: ～～
2. △△△（自信度: 15%）— 根拠: ～～
3. □□□（自信度: 5%）— 根拠: ～～
```

---

### Chain of Thought（段階的思考）

「ステップごとに考えて」と指示することで、推論の質が向上します。

```
以下の症例について、ステップごとに考えてから最終診断を述べてください。

ステップ1: 提示された画像所見を整理する
ステップ2: 所見から考えられる病態を列挙する
ステップ3: 臨床情報と照合して鑑別を絞る
ステップ4: 最も可能性の高い診断を結論づける

【症例】
45歳女性。頭痛と視野障害。
頭部造影MRIでトルコ鞍上部に径30mmの腫瘤。
T1WI等信号、T2WI高信号、均一に造影される。
視交叉を上方から圧排している。
```

---

### Few-shot（例示型）

入力と出力のペアを提示して、同じ形式で回答させます。

```
以下の例に倣って、画像所見を英語に翻訳してください。

例1:
入力: 「肝右葉に径2cmの嚢胞を認める」
出力: "A 2-cm cyst is noted in the right hepatic lobe."

例2:
入力: 「両側胸水を少量認める」
出力: "Small bilateral pleural effusions are present."

問題:
入力: 「膵頭部に径15mmの低吸収域を認め、主膵管の拡張を伴う」
出力:
```

### 制約付きプロンプト

不要な出力を防ぎ、必要な情報だけを得るための書き方です。

```
以下の条件に従って回答してください：
- 箇条書きで回答すること
- 各項目は2行以内にすること
- エビデンスレベルが低い情報には「※要確認」を付けること
- 患者への直接的な診断告知に使える表現は避けること

質問: 肺のすりガラス影（GGO）が見つかった場合、考えられる原因は何ですか？
```

---

### PREP法（Point / Reason / Example / Point）

Point（要点）→ Reason（理由）→ Example（具体例）→ Point（要点の繰り返し）で構成する説得力の高い文章構成法です。結論から述べるため短時間で伝わり、報告・プレゼン・文書作成に広く使えます。

```
以下のテーマについて、PREP法で説明文を作成してください。

【P（Point / 要点）】まず結論を1〜2文で述べてください
【R（Reason / 理由）】その結論に至る理由・根拠を述べてください
【E（Example / 具体例）】理由を裏付ける具体的な事例やデータを示してください
【P（Point / 要点の繰り返し）】最後に結論を再度述べ、聞き手の行動を促してください

テーマ: 放射線科における読影レポートの構造化（テンプレート化）を推進すべきである
対象: 部門会議での5分間のプレゼンテーション
```

### PDCAサイクル

Plan（計画）→ Do（実行）→ Check（評価）→ Act（改善）の4段階で考えさせるフレームワークです。業務改善やプロセス設計に向いています。

```
放射線科の読影レポート作成業務について、PDCAサイクルに沿って改善案を提案してください。

【Plan（計画）】現状の課題と改善目標を整理してください
【Do（実行）】具体的な改善施策を3つ提案してください
【Check（評価）】各施策の効果をどう測定するか示してください
【Act（改善）】評価結果をもとに次のサイクルで何を見直すべきか述べてください

現状の課題：
- レポート作成に1件あたり平均15分かかっている
- テンプレートが統一されておらず、記載漏れが発生する
- 緊急所見の伝達が遅れることがある
```

### SWOT分析

Strengths（強み）・Weaknesses（弱み）・Opportunities（機会）・Threats（脅威）の4象限で整理するフレームワークです。導入検討や意思決定に役立ちます。

```
放射線科の日常業務にローカルLLMを導入することについて、SWOT分析を行ってください。

以下の4項目それぞれについて、3つずつ挙げてください：

- Strengths（強み）: ローカルLLM導入の利点
- Weaknesses（弱み）: 導入における課題・限界
- Opportunities（機会）: 今後期待できる発展・活用
- Threats（脅威）: 注意すべきリスク・外部要因

最後に、分析結果を踏まえた総合的な導入推奨度を述べてください。
```

### 6つの思考の帽子（シックスハット法）

Edward de Bono が提唱した思考法で、6つの視点から1つのテーマを多角的に分析します。

| 帽子 | 視点 |
|------|------|
| 白 | 客観的事実・データ |
| 赤 | 感情・直感 |
| 黒 | 批判・リスク |
| 黄 | 楽観・メリット |
| 緑 | 創造・新しいアイデア |
| 青 | 全体の統括・結論 |

```
「AI画像診断支援ソフトを放射線科に導入すべきか」について、6つの思考の帽子（シックスハット法）で分析してください。

🎩 白い帽子（事実）: AI画像診断の現状の精度や導入実績のデータ
🎩 赤い帽子（感情）: 放射線科医として感じる期待や不安
🎩 黒い帽子（批判）: 導入のリスクや問題点
🎩 黄色い帽子（楽観）: 導入によるメリットや成功の可能性
🎩 緑の帽子（創造）: 従来にない新しい活用方法のアイデア
🎩 青い帽子（統括）: 全体を踏まえた結論と推奨アクション

各帽子について3〜5行で分析してください。
```

### 無限議論プロンプト

1つのLLMに複数の専門家を演じさせ、互いに議論させることで多角的な結論を導く手法です。水平思考を促し、一人では思いつかない視点が得られます。

**基本形：**

> あなたは{司会}、{専門家A}、{専門家B}の役割を持っています。今から質問者の{テーマ}について、呼び名で交互に発話させ課題と解決方法も混ぜながら、水平思考を使い議論してください。{ゴール}に向かって議論してください。かならず{ゴール}について結論を出してください。

**放射線科での活用例：**

```
あなたは{司会}、{専門家A}、{専門家B}の役割を持っています。今から質問者の{テーマ}について、呼び名で交互に発話させ課題と解決方法も混ぜながら、水平思考を使い議論してください。{ゴール}に向かって議論してください。かならず{ゴール}について結論を出してください。

各条件は以下の通りです。

#条件

{司会}：呼び名「田中教授」：放射線科の部門長。20年の臨床経験を持ち、業務効率と診断品質のバランスを重視する。

{専門家A}：「山本医師」：AI画像診断に詳しい若手放射線科医。テクノロジー導入に積極的で、最新論文に精通している。

{専門家B}：「佐藤医師」：ベテラン読影医。年間1万件以上の読影実績があり、AI依存のリスクや教育面への影響を懸念している。

{テーマ}
・放射線科の読影業務にAI支援ツールを導入すべきか。導入する場合、どのような運用ルールが必要か。

{参考資料}
無し

{ゴール}
・AI読影支援の具体的な導入計画（対象モダリティ、運用フロー、教育体制、評価基準）を策定し、3名が合意する結論を出す。
```

**議論を続けたい場合：**

LLMが結論を出した後も、以下のように追加の論点を投げることで議論を継続できます。

```
ありがとうございます。ではさらに以下の論点について、3名で議論を続けてください：
・導入後に読影精度が低下した場合の撤退基準
・研修医の教育プログラムへの影響
```

### 比較: 同じ質問でのプロンプトの違い

以下の3つを順番に試して、回答の違いを比較してみてください。

**A. シンプルなプロンプト:**
```
肺塞栓症について教えてください。
```

**B. RTFフレームワーク:**
```
【Role】あなたは放射線診断専門医です。
【Task】肺塞栓症について、画像診断の観点から解説してください。
【Format】CT所見・MRI所見・鑑別診断の3項目に分けてください。
```

**C. RICE + 制約付き:**
```
【Role】あなたは放射線科レジデント向けの教育担当医です。

【Context】対象はCT読影を始めたばかりの初期研修医です。

## Tips
