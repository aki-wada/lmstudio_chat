# Google Forms でアンケートを作成する方法

## 1. Google Forms にアクセス

- [forms.google.com](https://forms.google.com) にアクセス
- 「空白のフォーム」を選択

---

## 2. 質問タイプの対応表

| アンケート項目 | Google Forms の質問タイプ |
|---------------|-------------------------|
| Q1 年齢 | **ラジオボタン**（単一選択） |
| Q2 デバイス選択 | **ラジオボタン** |
| Q2-A/B 詳細 | **セクション分岐** + ラジオボタン |
| Q3 クラウドLLM | **チェックボックス**（複数選択） |
| Q3 使用頻度 | **ラジオボタン** |
| Q3 使用目的 | **チェックボックス** |
| Q4 モデル | **ラジオボタン** + **チェックボックス** |
| Q5 用途と点数 | **グリッド形式**（後述） |
| 機能満足度 | **均等目盛り** or **選択式グリッド** |
| Q10 比較評価 | **記述式（短文）** |
| 自由記述 | **段落**（長文回答） |

## 3. セクション分岐の設定（Q2）

Q2で「Mac」と「Windows」を選択した場合に、それぞれ異なる詳細質問に分岐させる方法：

1. Q2で「Mac」「Windows」「その他」を選択肢として作成
2. 質問の右下「︙」（3点メニュー）をクリック
3. 「回答に応じてセクションに移動」を選択
4. 各選択肢の横に移動先セクションを設定：
   - Mac → Q2-Aセクション
   - Windows PC → Q2-Bセクション
   - その他 → Q3セクション（スキップ）

### セクション構成例

```
セクション1: 基本情報（Q1）
    ↓
セクション2: デバイス選択（Q2）
    ├─ Mac → セクション3: Mac詳細（Q2-A）
    ├─ Windows → セクション4: Windows詳細（Q2-B）
    └─ その他 → セクション5: クラウドLLM（Q3）
    ↓
セクション5: クラウドLLM（Q3）
    ↓
（以下続く）
```

## 4. Q5の用途別点数の作成方法

### 方法A: 選択式グリッド（推奨）

1. 質問タイプで「選択式（グリッド）」を選択
2. 行に各用途を入力：
   - 医学辞書として（用語の意味、疾患の説明など）
   - 画像診断（読影支援、症例検討）
   - 論文の解説・要約・翻訳
   - 文書・プレゼン作成（レポート、サマリーなど）
   - 質問・調査（情報収集など）
   - 画像の分析（Vision機能）
3. 列に点数範囲を入力：
   - 試していない
   - 0-20点
   - 21-40点
   - 41-60点
   - 61-80点
   - 81-100点

### 方法B: 各用途を個別質問に分割

各用途について2つの質問を作成：

```
Q5-1. 医学辞書として試しましたか？
  - はい
  - いいえ

Q5-1b. （はいの場合）点数を入力してください（0-100）
  [記述式（短文）]
```

## 5. 機能満足度の作成方法

### 均等目盛りを使用

1. 質問タイプで「均等目盛り」を選択
2. 1〜5の目盛りを設定
3. ラベルを設定：
   - 1 = 不満
   - 5 = 満足

### または選択式グリッド

1. 行に各機能を入力
2. 列に「1（不満）」「2」「3（普通）」「4」「5（満足）」「N/A」を入力

## 6. 効率的な作成手順

### Step 1: フォームの基本設定

1. タイトル: 「Local LLM Chat ユーザーアンケート」
2. 説明文を追加（任意）

### Step 2: セクションを先に作成

以下の順番でセクションを追加：

1. 基本情報
2. 使用環境（デバイス選択）
3. 使用環境（Mac）
4. 使用環境（Windows）
5. クラウドLLM
6. ローカルLLM
7. 用途と評価
8. 機能の満足度
9. UI/デザイン
10. 改善要望
11. 医療現場での利用
12. 自由記述

### Step 3: 各セクションに質問を追加

SURVEY.md の内容を参照しながら、各セクションに質問を追加

### Step 4: 分岐設定

Q2の回答に応じてMac/Windowsセクションに分岐するよう設定

### Step 5: プレビューでテスト

右上の「プレビュー」ボタンで動作確認

---

## 7. 便利な機能

### 必須回答の設定

重要な質問は「必須」をONにする（質問右下のトグル）

### 説明文の追加

質問の下に補足説明を追加可能（「︙」→「説明」）

例：
- Q2-A: 「確認方法: 画面左上の  メニュー → このMacについて」
- Q4: 「確認方法: LM Studio 画面上部のモデル選択欄」

### 画像の追加

確認方法のスクリーンショットを添付すると分かりやすい

### その他オプションの追加

チェックボックス/ラジオボタンの選択肢に「その他」を追加：
- 選択肢の下にある「その他を追加」をクリック

---

## 8. 回答の集計

### スプレッドシートへのエクスポート

1. 「回答」タブをクリック
2. 右上のスプレッドシートアイコンをクリック
3. 「新しいスプレッドシートを作成」を選択
4. 自動でGoogle Sheetsに回答が集計される

### 回答の概要表示

「回答」タブの「概要」で、グラフ形式で回答を確認可能

---

## 9. 共有設定

### フォームの共有

1. 右上の「送信」ボタンをクリック
2. リンクを取得（短縮URLオプションあり）
3. メールで送信も可能

### 回答の制限

- 「設定」→「回答」で以下を設定可能：
  - メールアドレスの収集
  - 回答を1回に制限
  - 回答後の編集を許可
